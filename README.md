## 🧠 Overview
The **AI-Powered Patient Safety and Clinical Decision Support System (PSACDSS)** is a modern healthcare dashboard designed to help medical professionals:

- Monitor patients in real-time  
- Predict clinical risks using AI  
- Manage prescriptions safely  
- Respond to critical alerts instantly  

This system simulates **AI-driven decision support** to improve patient safety and reduce medical errors.

---

## ✨ Key Features

### 🏥 Patient Management
- List of active patients with risk indicators  
- Detailed patient profiles (history, vitals, medications)  
- Clean and structured data view  

---

### 🧠 AI Clinical Decision Support (CDSS)
- 📊 Risk prediction (Sepsis, Pulmonary Embolism)  
- 💡 AI-based treatment recommendations  
- 📈 Confidence score for decisions  

---

### 🚨 Live Alert Center
- Real-time alerts for critical conditions  
- Severity levels:
  - 🔴 Critical  
  - 🟡 Warning  
  - 🔵 Notice  
- Instant response system  

---

### 💊 Prescription Safety
- Drug interaction detection  
- Allergy checking  
- AI-assisted validation  

---

### 📈 Vitals Monitoring
- ❤️ Heart Rate  
- 🩸 Blood Pressure  
- 🫁 Oxygen (SpO2)  
- 🌡️ Temperature  
- Interactive charts using Recharts  

---

### 👥 Users & Roles
- 👨‍⚕️ Doctors  
- 👩‍⚕️ Nurses  
- 🛠️ Admins  
- Role-based access control
